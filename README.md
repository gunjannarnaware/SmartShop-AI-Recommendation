# Smart Recommendation System for E-Commerce
**Developed by: Gunjan**

## Project Overview
This is an AI-powered e-commerce platform that uses **Similarity-based learning** to suggest products to users.

## Technical Details
- **AI Engine:** TF-IDF Vectorization & Cosine Similarity
- **Backend:** Python Flask
- **Database:** User-Item Matrix via products.csv
- **UI:** Modern Lavender & Beige Aesthetic

## How to Run Locally
1. Download this repository as a ZIP.
2. Open the folder in VS Code.
3. Install dependencies: `pip install flask pandas scikit-learn`.
4. Run the app: `python app.py`.
5. Open `http://127.0.0.1:5000` in your browser.
